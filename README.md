# Responsive Website Project — HTML5 & CSS3

## Overview
This project is a fully responsive and polished website created using HTML5 and CSS3. The purpose of the assignment was to apply modern web development techniques including semantic structure, media queries, flexbox design, interactive styling, and visual polish.

This README documents the development workflow, GitHub hosting configuration, and local server deployment using XAMPP.

---

## Features
- ✅ Fully responsive layout using media queries  
- ✅ Modern styling with CSS3  
- ✅ Semantic HTML5 structure  
- ✅ Flexbox-based layout organization  
- ✅ Mobile-first design  
- ✅ GitHub Pages hosting  
- ✅ Local server testing using XAMPP
